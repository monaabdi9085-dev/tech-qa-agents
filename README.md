# Multi-agent Tech Q&A

I detta projekt har jag skapat ett multi-agent system bestående av tre agenter: Planner, Doer och Critic.

Systemet är utformat för ett företag som tidigare endast  arbetat med “papper och penna” och nu tar steget in i den digitaliserade arbetsmiljön. Detta kommer självklart inte vara lätt och problem
kommer naturligtvis uppstå!. 

Agenterna samarbetar för att:
- förstå användarens problem (Planner)
- ta fram enkla steg-för-steg lösningar (Doer)
- granska och verifiera lösningarna (Critic)

Målet är att ge tydliga och lättförståeliga instruktioner till användare utan tidigare datorvana.
